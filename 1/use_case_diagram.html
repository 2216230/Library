üèóÔ∏è System Architecture Overview

  • Framework: AdminLTE 3.3.7 with Bootstrap 3.3.7
  • Backend: PHP with prepared statements and MySQLi
  • Database: MySQL/MariaDB with 15+ tables
  • Authentication: Session-based with role hierarchy (Admin/Student/Faculty)
  • Email: PHPMailer for notifications and alerts
  • E-Books: Calibre integration for digital collection
  • Reporting: Word/PDF export functionality
  • Security: Prepared statements, password hashing, session validation
